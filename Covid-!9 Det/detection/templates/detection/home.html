Welcome to Dorsel Hand Vein Authentication

Your go-to platform for Dorsel hand vein Based Authentications.

Features

Our application provides secure access with user authentication, allowing you to log in and log out securely. Upload Dorsel Vein images and receive immediate predictions using our pre-trained TensorFlow model. Manage patient details efficiently with secure storage and easy retrieval based on specific criteria.

Use the patient lookup feature to find patient details and past scan results quickly. The application is built with Django for rapid development and TensorFlow for accurate detection, and uses SQLite3 for reliable data management.

Technology Stack

We use Django for backend development, TensorFlow for deep learning, and SQLite3 for data storage. This combination ensures a robust, scalable, and efficient application.

How It Works

Start by logging in, upload a Dorsel Hand Vein Image, and click "Predict" to get immediate results. Manage patient details, add new records, and view past scan results with ease.
