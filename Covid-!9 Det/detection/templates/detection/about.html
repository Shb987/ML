About This Application

Discover how our Dorsel hand Vein Authentication Application leverages advanced technology to provide accurate and reliable results from Dorsel Veins.

User Authentication

  • Secure Login: Users are authenticated using credentials stored securely in a configuration file. This ensures that only authorized personnel can access the application.
  • Logout Functionality: Users can securely log out of the application to prevent unauthorized access.

Database Connectivity

  • SQLite3 Database Integration: The application connects to an SQLite3 database to store and retrieve patient information. This allows for efficient management of patient data within a lightweight and self-contained database.
  • Cached Connection: To enhance performance, the database connection is cached, ensuring quick access to data without repeatedly establishing new connections.

Covid-19 Detection

  • Pre-trained Model: We utilize a pre-trained TensorFlow model to detect Dorsel Hand Vein Images. This model has been trained on a large dataset to ensure high accuracy.
  • Image Processing: The application preprocesses the uploaded images to fit the input requirements of the model, ensuring accurate predictions.

Patient Management

  • Data Entry: Users can input patient details such as name, age, and sex. The application ensures all fields are filled before processing to prevent incomplete data entries.
  • Data Storage: The details of each patient, along with their CT scan image and prediction result, are stored in the SQLite3 database.
  • Filtering: Users can filter and view patient data based on specified criteria, making it easier to manage and retrieve patient records.

Prediction and Results

  • Image Upload: Users can upload chest CT scan images, which are then displayed within the application.
  • Prediction: By clicking the "Predict" button, the application processes the uploaded image and displays whether the scan indicates Covid-19 or not.
  • Guidance: Based on the prediction, the application provides recommendations on next steps, such as isolating and consulting healthcare providers for positive cases.

Viewing Results

  • Patient Lookup: Users can enter the name and age of a patient to view their details and the results of previous scans.
  • Detailed Results: The application retrieves and displays comprehensive patient details from the SQLite3 database, including the latest scan results.

Technology Stack

  • Django: The application is built using Django, a high-level Python web framework that encourages rapid development and clean, pragmatic design.
  • TensorFlow: The deep learning model for Covid-19 detection is implemented using TensorFlow.
  • SQLite3: A lightweight, self-contained SQLite3 database is used for storing and managing patient data.
